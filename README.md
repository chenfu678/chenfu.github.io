### 4.使用方法

##### 4.1起始时间

请在`index.html`中的计时器模块自行修改。相差了一个月，故真实时间往前填一个月

##### 4.2背景图

在`index.html`中修改：

```js
.wrap {
	background-image: url(https://bing.img.run/rand.php);
}
```
